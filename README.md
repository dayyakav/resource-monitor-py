## Simple Resource Monitor

### The only reason this project exists is my college graduation project. Please don't tell me anything about code quality..

Simple CLI resource monitor written in Python using psutil.

# Screenshots


# Features

Displays:

- CPU usage
- Memory usage (used and total)
- Network info (download and upload)
- Running processes (PID, process name, CPU usage)

# Requirements

- [Python 3.10+](https://www.python.org/downloads/)
- Installed [uv](https://docs.astral.sh/uv/)

# Installation

1.  Download source code from project page
    
2.  Move to the project folder
    
3.  Run:
    
    ```Python
    uv run main.py
    ```
    

This will install dependencies and run project.
